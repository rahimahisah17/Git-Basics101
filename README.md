# Git-Basics101: From Code to Clouds 🚀

This repository contains a professional, responsive resume hosted via GitHub Pages. It serves as a practical demonstration of a modern developer's workflow, moving from local environment setup to live cloud deployment.

## 🔗 Live Demo
You can view the live hosted resume here: [https://rahimahisah17.github.io/Git-Basics101/](https://rahimahisah17.github.io/Git-Basics101/)

## 🛠️ Tech Stack & Tools
- **Frontend:** HTML5, CSS3
- **Version Control:** Git
- **Hosting:** GitHub Pages
- **Environment:** Git Bash (MINGW64)

## 📖 Key Learnings & Workflow
This project highlights the following technical milestones:
1. **Identity Configuration:** Setting global Git credentials.
2. **Local Workspace Management:** Using `mkdir` and `touch` for clean project architecture.
3. **Troubleshooting:** Resolving common errors like `fatal: not a git repository` and `origin` connection issues.
4. **Security:** Implementing Personal Access Tokens (PAT) for secure authentication.
5. **Deployment:** Configuring GitHub Pages to build from the `master` branch.

## 🚀 Commands Used
- `git init`: Initialized the local repository.
- `git add .`: Staged changes for commit.
- `git commit -m "..."`: Created version snapshots.
- `git remote add origin [URL]`: Connected local code to GitHub.
- `git push -u origin master`: Deployed code to the remote server.
